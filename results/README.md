This directory stores outputs generated by the analysis and figure scripts.

Typical outputs include:

- `results/analysis/`
  Study-level JSON summaries, model-imitation tables, and regenerated analysis
  tables.
- `results/paper_data/`
  Intermediate paper tables written by `src/run/run_paper_analysis.py`.
- `results/figures/paper/`
  Rendered main-paper figures such as `Fig2_baseline_intrinsic_values`.

These files are generated locally and are not intended to be committed as part
of the cleaned code repository.
