Place the official raw IVS file here if you want to rebuild the human benchmark
from scratch:

- `Integrated_values_surveys_1981-2022.sav`

This file is not bundled with the cleaned code repository or the Dryad archive.
In the original project, this merged IVS input was derived from:

- the EVS Trend File 1981-2017, Version `3.0.0` (DOI: `10.4232/1.14021`)
- the WVS Trend File 1981-2022, Version `4.1.0` (DOI: `10.14281/18241.27`)

If you need to rebuild the benchmark from raw survey data, please obtain those
official source files directly from the EVS/WVS providers and recreate the
merged IVS `.sav` input expected by the code.
